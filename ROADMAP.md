## Details



### 0.1.x — First public release

Multi-provider fonts, CLI tools, Flex recipe, local lock workflow

### 0.2.x — Pairing and exports

Font pairing import, expanded export formats, migration guides

### 0.3.x - Font Intelligence

Coverage, Glyphs, Sprache, License-Metadaten

### 0.4.x - Typography System

Font stacks, scales, line-height, responsive typography

### 0.5.x - Production Optimization

Subsetting polish, preload, unicode-range, reports

### 0.6.x - Provider Hardening

Provider contracts, cache, fallbacks, offline behavior

### 0.7.x - DX / Doctor

fonts:doctor, diagnostics, config repair, explain output

### 0.8.x - Enterprise / Brand Fonts

private registries, authenticated downloads, license audit

### 0.9.x - API Freeze

BC layer, deprecations, docs, cookbook, final migration guide

### 1.0.x - Stable API

Stable provider/export/config contracts, export policy, and handbook
